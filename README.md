# Sample: List Group using ThinReports

## How to run

Install ThinReports:

    gem install thinreports

Then, execute people.rb:

    ruby people.rb

Please see people.pdf.

## Requirements

  * Ruby > 1.9
  * ThinReports >= 0.7.5
